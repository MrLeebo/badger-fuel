=======
badger-fuel
===========

Organize lunch orders at the office

====
Installation
====

```
# Clone the Repo
git clone https://github.com/MrLeebo/badger-fuel.git
cd badger-fuel

# Install Meteor and Meteorite
curl https://install.meteor.com | /bin/sh
npm install -g meteorite
mrt install

# Run the Server
meteor
```
